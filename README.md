# Marketing Digital Academy

## Descripción del proyecto

Marketing Digital Academy es una aplicación web desarrollada como proyecto de la asignatura **Desarrollo Web**. Su objetivo es promocionar y ofrecer cursos de Marketing Digital mediante una interfaz moderna, interactiva y fácil de usar.

La aplicación permite a los usuarios conocer los cursos disponibles, consultar sus beneficios, visualizar testimonios y enviar un formulario de contacto para solicitar información o realizar una inscripción.

---

## Objetivo

Desarrollar una aplicación web utilizando HTML, CSS y JavaScript, aplicando buenas prácticas de desarrollo, organización del proyecto y control de versiones con Git y GitHub.

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* Visual Studio Code

---

## Funcionalidades

* Página principal con diseño responsivo.
* Menú de navegación.
* Presentación de la academia.
* Catálogo de cursos.
* Sección de beneficios.
* Testimonios de estudiantes.
* Formulario de contacto con validación.
* Botones de compra con interacción mediante JavaScript.
* Botón para volver al inicio de la página.
* Actualización automática del año en el pie de página.
